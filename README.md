# Low-resource Voice Conversion GAN 
This is a voice conversion project aims at 109 speakers with limited training samples.

[Paper](https://arxiv.org/abs/2010.11646) is submitted to ICASSP 2021 
### Requirements:
 1. Python3
 2. PyTorch 0.4.1
 3. Pyworld


### Preprocess
```
./run_pre.sh
```

### Train:
```
./run_train_adain.sh
```

### Convert
```
./run_convert.sh
```
